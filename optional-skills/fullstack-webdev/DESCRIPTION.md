# Full-Stack Web Development

Comprehensive skills for professional full-stack web developers — from project scaffolding to production deployment.

## Skills

### Project Setup & Architecture
- **project-scaffolding** — T3 Stack, Next.js Full-Stack, MERN, Next.js+FastAPI, monorepo templates
- **monorepo-patterns** — Turborepo/Nx workspace setup, shared packages, CI optimization

### Core Development
- **typescript-mastery** — Generics, utility types, discriminated unions, tsconfig, type inference
- **react-patterns** — Server Components, TanStack Query, Zustand, form handling, Suspense
- **tailwind-design-system** — Design tokens, dark mode, responsive utilities, animation
- **shadcn-ui** — Component library patterns, form integration, theming, data tables

### Styling & Design
- **responsive-design** — Mobile-first, container queries, fluid typography, viewport units

### API & Backend
- **api-design** — REST conventions, GraphQL, tRPC, OpenAPI, pagination, error shapes
- **nodejs-backend-patterns** — Express structure, async errors, BullMQ, security middleware
- **redis-cache** — Cache-aside, sessions, rate limiting, pub/sub, distributed locks
- **prisma-expert** — Schema design, migrations, transactions, raw queries, performance

### Database & Storage
- **server-management** — PM2, systemd, Nginx/Caddy reverse proxy, Docker Compose

### Testing & Quality
- **testing-qa** — Vitest, Playwright E2E, Lighthouse CI, test coverage, mocking
- **sentry-integration** — Error tracking, performance monitoring, source maps, React Error Boundaries
- **auth-implementation-patterns** — JWT, sessions, OAuth2, Argon2, RBAC, secure cookies

### Deployment & DevOps
- **docker-expert** — Multi-stage builds, layer caching, Docker Compose dev/prod, hardening
- **github-actions-templates** — CI/CD pipelines, Playwright, Docker, Lighthouse, Dependabot
- **vercel-deployment** — Next.js deployment, edge functions, preview URLs, cron jobs
- **cloudflare-workers** — D1, KV, R2, Durable Objects, edge computing patterns
- **nginx-caddy** — Reverse proxy, SSL/TLS, load balancing, security headers

## When to Load These

Load relevant skills before starting a full-stack web project or when working on specific areas:
- Building a new project → project-scaffolding + typescript-mastery + react-patterns
- Designing an API → api-design + auth-implementation-patterns
- Setting up CI/CD → github-actions-templates + docker-expert
- Testing → testing-qa + sentry-integration
- Deploying → vercel-deployment or docker-expert or cloudflare-workers
